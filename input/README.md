# Input Directory

All blocks refer to chromosome 19.

## Folders
* **1000G**: 1000 Genomes data for Block 37 
* **annovar**: annovar output for Block 37 (needed for creating the master legend files)
* **gnomad**: gnomAD allele counts and downsampling data for Block 37
* **mac_bins**: target number of variants (nvar) and allele frequency spectrum (AFS) data for Block 37
* **positions**: gencode coding positions for each block (consistent across ancestries)

## Files
* **chr19.block37.*.master.legend**: master legend file with annotation information for all known and unknown coding positions (one file for each population)
